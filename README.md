# ALX Listing App

## Project Description
ALX Listing App is an Airbnb clone project focused on creating a responsive and user-friendly property listing page. This project demonstrates modern web development practices using Next.js, TypeScript, and Tailwind CSS.

## Project Goals
- Build a responsive Airbnb-style listing page
- Implement reusable React components
- Ensure type safety with TypeScript
- Create a maintainable and scalable codebase

## Project Structure

### `components/`
Contains reusable React components:
- `common/Card.tsx` - Display property information
- `common/Button.tsx` - Reusable button component

### `interfaces/`
TypeScript interface definitions for type safety across the project.

### `constants/`
Application constants, configuration settings, and reusable text strings.

### `public/assets/`
Static assets including images, icons, and other media files.
